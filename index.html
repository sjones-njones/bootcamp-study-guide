HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Always begin HTML file with a declaration DOCTYPE html, which is not case sensitive followed by lang = "en".

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Wildcard symbol - * All elements will have the same
  • Inline CSS - styles element directly in HTML file, quick and permanent but not flexible
  • Internal style sheet - CSS rules in head section, rules only apply to that page
  • External CSS stylesheet - most common, easier to change
  • Two selectors can be combined using a comma.
  • DRY coding - do not repeat yourself code
  • Img will refer to all images
  • Display:block; puts each image on a new line.
  • .card - the period designates class sector - any HTML assigned card will gain

Git

  • Git commands do not = bash commands and always start with GIT.
  • Git tracks changes locally.
  • Git add -A: saves staged files
  • Git status: checks what branch we are currently on
  • Git checkout -b branch-name: creates a new branch and switches to it
  • Git checkout name: switches branches

Gitbash CLI Command

  • CLI: command line interface-allows functionality for sites through texts
  • GUI: graphical user interface
  • Pwd: print working directory
  • Ls: list of files in working directory
  • Cd name: change directory
  • Cd ..: moves back one level
  • Mkdir name: make directory (folder)
  • Touch name.txt: creates text file
  • Start name: opens file

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script. Js reads from top to bottom and order counts. Conditional statements and loops control the order.
  • To define a variable, var name = "string";
  • Console.log method - output message to web by adding argument, use console.log(name);
  • To link JS to HTML use script src= "link" in body (not in head like we did with CSS)
  • If(condition){ //block of code to be executed if condition is true } else if (second condition) { //block ex if 2nd condition is true }
  • Arrays contain a set of data
    • To define an array use var name = [array values can include strings, numbers, and Boolean values];
  • Console.log(name[0]); will log the first data point in array
  • "for" loop you need:
    • A starting point var x=0; this tells loop to start at beginning of array
    • A condition - as long as x is less than length of array, loop will still run and execute each item x 'less than sign' name.length
    • A final statement - x++ is the same as x+1, everytime the code loops, x will increase by 1
    • for (var x = 0 ;x 'less than sign' name.length;x++) {// code block}
  • Commenting out code - makes code inactive w/o deleting it /* */
  • Functions - set of instructions
    • Define function: function maketea() {//code block}
    • Call function: maketea()
    • Always indent for loop inside function
  • Random topic randomizes selection
